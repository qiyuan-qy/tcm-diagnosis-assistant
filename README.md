# TCM智能诊疗系统

中医智能诊疗辅助系统，基于FastAPI + Vue开发。

## 功能特性

- 🩺 智能问诊
- 💾 病例管理
- 📊 数据分析
- 🔧 系统管理

## 技术栈

### 后端
- Python 3.10+
- FastAPI
- SQLAlchemy
- PostgreSQL

### 前端
- Vue 3
- Element Plus
- Axios

## 快速启动

\`\`\`bash
# 启动系统
./start.sh
\`\`\`

访问: http://localhost

## 项目结构

\`\`\`
tcm-diagnosis-assistant/
├── backend/         # 后端代码
├── frontend/        # 前端代码
├── nginx/          # Nginx配置
├── data/           # 数据文件
└── start.sh        # 启动脚本
\`\`\`
